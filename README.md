# Credit Risk Analysis (SQL Project)

## Objective
Analyse customer financial risk using SQL by studying income, age, and debt patterns.

---

## Analysis Performed

- Risk Distribution (Low, Medium, High)
- High Risk Customer Profile
- Age vs Risk Analysis
- Income vs Risk Analysis

---

## Key Insights

- Low income customers are most likely to be high risk  
- High income customers are mostly low risk  
- Adults form the largest risk group  
- Younger customers show relatively higher financial risk  

---

## Tools Used

- SQL (BigQuery)
- Data Cleaning (SAFE_CAST)
- Aggregations & CASE statements

---

##  Project Files

- `credit_risk_analysis.sql` → Contains all queries used in the analysis

---

## Outcome

This project demonstrates how SQL can be used to extract meaningful financial insights and identify high-risk customer segments.
